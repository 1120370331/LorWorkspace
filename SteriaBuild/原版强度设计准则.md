# 基于原版整理的角色强度设计准则

> 以都市之星中上层为准则

---

## 一、被动技能

### 被动费用等效表

| 费用 | 等效效果 |
|------|----------|
| **1费** | 开局多1张书页 |
| **2费** | 1层非强壮/忍耐的正面BUFF |
|        | 条件下1层强壮/忍耐（如拥有10层充能、情感等级提升等） |
|        | 1点额外增伤（条件下可到3） |
|        | 1点额外减伤（条件下可到3） |
|        | 某种DEBUFF施加层数+1（仅限流血、烟气） |
| **3费** | 条件下1点光芒回复 |
|        | 条件下2层强壮或忍耐 |
| **4费** | 条件下骰子威力+1/强壮（如指定骰子类型、经过3幕、使用6张书页等） |
|        | 调整1级抗性（敌方或自身） |
|        | 2层非强壮/忍耐的正面BUFF |

## 二、战斗书页

### 骰子威力基准

> **说明**：
> - 骰子总数是一般能达到的骰子数量
> - 超过此数量：多的骰子需要削弱总点数，且削弱后点数应小于平均点数（多骰强度 > 单骰）
> - 少于此数量：骰子威力基准可以略微提高

| 费用 | 骰子数 | 进攻型 | 防御型(格挡) | 防御型(闪避) | 反击型(攻击) | 反击型(防御) |
|------|--------|--------|--------------|--------------|--------------|--------------|
| **0费** | 1 | 3-8 | 3-8 | 3-8 | - | - |
| **1费** | 2 | 4-8 | 4-8 | 4-8 | - | - |
| **2费** | 3 | 4-8 | 4-8 | 4-8 | 3-7 | - |
| **3费** | 3.5 | 5-9 | 6-9 | 4-8 | 3-7 | 3-8 |

---

### 书页效果费用等效

> **说明**：
> - 费用等效是指在某光芒消耗下，一个正常骰子威力基准的书页能够额外产生的效果
> - 若书页没有额外效果，可以基于基准提高威力
> - 若书页效果超过费用基准，应当适度降低威力
> - **免费效果规则**：N费书页可以免费包含1个0费效果（如2费充能卡可获得3+3充能，1费抽牌卡可抽1+1张）

| 费用 | 等效效果 |
|------|----------|
| **0费** | 抽1张书页 |
|        | 回1点光芒 |
|        | 3层充能 / 2层烟气 |
| **1费** | 抽1张书页 |
|        | 回1点光芒 |
|        | 施加流血+1 |
| **2费** | 3层充能 / 4层烟气 / 3层命中烧伤 |

> **注**：2费及以上书页可以拥有独特的效果，考虑到体系和角色人设较为复杂，不再做统一合模统计。
