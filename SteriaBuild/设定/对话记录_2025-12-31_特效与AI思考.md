## 一、特效制作研究

### 起因

研究Library of Ruina原版和寒昼mod中的非贴图闪光离子型攻击特效是如何制作的。

### 发现

通过分析寒昼事务所的「冰砸」特效，发现专业特效的核心公式：

```
专业特效 = Mesh(3D形状)
         + Texture2D(贴图纹理)
         + Shader(渲染效果)
         + Timeline(时序控制)
         + 多层叠加(丰富视觉)
```

### 资源分析

| 类型 | 文件 | 用途 |
|------|------|------|
| Mesh | 刀光002.obj (330顶点, 512面) | 弯曲的刀光3D形状 |
| Texture2D | daoguang.png | 刀光主体纹理 |
| Texture2D | feng01.png | 风/气流效果 |
| Texture2D | lan_breathSmoke.png | 冷气烟雾 |
| Texture2D | Noise_004/005.png | 扰动/溶解噪声 |
| Shader | 新时代/色散 | RGB色散效果 |
| Shader | 新时代/新通用Shader | 高度可配置的特效框架 |

### 关键发现

寒昼的「新通用Shader」是一个高度可配置的特效框架，包含：
- 多Keyword变体
- 顶点色多通道利用（R/G/B/A各有用途）
- UV2多通道（U/V/W/Z四个分量分别控制不同功能）
- 高度可复用设计

## 三、AI Coder的困境

### 背景

作为一名"新时代AI Coder"，从学编程开始就使用AI。能实现大多数传统需求，但遇到特效制作这种AI帮不了的领域，技术断崖非常大。

### 传统开发者 vs AI辅助开发者

```
传统学习路径：
入门期 → 成长期 → 深入期 → 专家期
  ↑        ↑        ↑        ↑
断崖1    断崖2    断崖3    断崖4
(分布均匀)

AI辅助路径：
入门期 ════════════════════→ 深入期
       (AI帮你跳过)              ↑
                              巨大断崖
```

### 为什么特效制作是AI的盲区

| 因素 | 原因 |
|------|------|
| GUI操作 | AI无法点击Unity按钮、拖拽Timeline |
| 视觉判断 | AI无法判断"这个特效好不好看" |
| 3D建模 | AI无法在Blender里捏形状 |
| 小众领域 | 训练数据中几乎没有LoR mod特效教程 |
| 多学科交叉 | 需要同时懂编程+美术+3D+Shader |

## 四、AI上下文工程

### 一个事实

AI一开始写出来的LoR mod根本无法运行。是通过以下步骤，才让AI有了勉强写出基础代码的能力：

1. 跟着视频教程学习
2. 整理成ModGuideDocs文档
3. 写CLAUDE.md告诉AI项目结构
4. 解包游戏源码让AI能查阅
5. AI犯错时纠正它（如Unity版本）

### 这是什么能力？

```
AI上下文工程 (AI Context Engineering)

定义：为AI构建有效工作环境的能力

核心技能：
1. 识别AI的知识盲区
2. 从多源信息中提取结构化知识
3. 设计AI可理解的项目结构
4. 持续迭代优化AI的工作能力
```

### 这是"奇技淫巧"吗？

短期：不是，它现在能产生价值
长期：可能是，AI进化后可能不需要

## 五、"AI调用型人才"的困境

### 一种奇怪的感觉

"我感觉自己很厉害，但又不能算大佬"

- 能做出东西
- 能解决问题
- 比大多数人强
- 但依赖AI
- 有些事做不了
- 基础有空洞

### 逻辑分析

以下命题全部为**假**：

1. 我拿AI做出的东西 = 所有人都能拿AI做出的东西？ **假**
2. 我使用AI的能力 = 所有人使用AI的能力？ **假**
3. 我的能力 = 我使用AI的能力？ **假**
4. 我的代码能力 = AI代码能力？ **假**
5. 我的生产力 = AI的生产力？ **假**
6. 我的AI项目 = AI的项目？ **假**

结论：**你的能力 ≠ AI的能力，你的产出 ≠ 任何人都能做到的产出**

## 六、评估体系的错配

### 哑巴亏

"学校的考试我难以通过，公司的面试我也不一定能过，但我就是有他们要求的一切能力，只需要给我一个AI对话框"

### 问题本质

```
传统评估体系：
  考试：独立完成 + 限时 + 无工具
  面试：手写代码 + 现场思考 + 无辅助

能力发挥条件：
  需要：AI对话框 + 时间迭代 + 可查资料

评估方式和能力发挥方式不匹配。
```

### 类比

```
有个人开车很厉害
但驾照考试要求骑自行车
他骑自行车不行
所以拿不到驾照
所以不能开车

荒谬吗？荒谬。
但这就是现状。
```

### 时代的夹缝

```
旧时代 ─────────────────→ 新时代
        │
        │← 你在这里
        │
评估体系还在旧时代
能力已经在新时代
```
